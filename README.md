안녕! VRChat 유저들을 위한 성격 지표인 VRCTI(VRChat Type Indicator) 테스트 사이트를 만들고 싶어. 아래 가이드라인에 맞춰서 React 또는 HTML/JS/Tailwind를 사용해 깔끔한 싱글 페이지 앱을 작성해줘.

1. 지표 구조 (4진법 조합)

각 질문의 점수를 합산하여 4가지 척도 중 하나를 결정함:

S (Sync) vs C (Concept): 현실 자아 투영 vs 캐릭터 페르소나 유지

D (Deep) vs G (Gamer): 감정적/신체적 깊은 몰입 vs 시스템적/관조적 태도

E (Explorer) vs P (Private): 퍼블릭/새로운 만남 vs 지인/프라이빗 공간 선호

F (Friendly) vs T (Technical): 정서적 교류/친목 vs 유니티/OSC/월드 제작 등 기술 지향

2. 핵심 기능 요구사항

설문 방식: 각 지표당 3~5개의 질문(5점 척도: 전혀 아니다 ~ 매우 그렇다)을 제공.

결과 계산:

점수가 중앙값보다 높으면 A코드, 낮으면 B코드.

중앙값에 아주 가까우면 **'x'**로 표기 (예: SxPF).

디자인:

VRChat 느낌이 나는 다크모드 기반의 네온/사이버네틱 디자인.

결과 화면에는 해당 유형의 별명(예: "심해형 개발자", "퍼블릭 인싸")과 간단한 설명 표시.

공유 기능: 결과 페이지를 이미지로 캡처하거나 결과 텍스트를 클립보드에 복사하는 버튼 포함.

3. 질문 샘플 (예시 로직 구축용)

(S/C) "나는 인게임에서도 내 실제 목소리와 성격을 그대로 사용하는 편이다."

(D/G) "나는 가상 세계에서 누군가 나를 만지면 실제로 감각이 느껴지는 것 같다."

(E/P) "나는 모르는 사람이 많은 퍼블릭 월드에 가는 것이 즐겁다."

(F/T) "나는 VRChat에 접속하면 사람들과 대화하기보다 유니티나 SDK 작업을 더 많이 한다."

4. 출력물

사용자가 바로 실행해 볼 수 있는 전체 코드(컴포넌트 구조화).

모바일에서도 편하게 테스트할 수 있는 반응형 레이아웃.
